Dr. Mariana Karar

Web Application Development (WAD) focusses on platform-independent and browser-independent languages such as HTML, JavaScript and CSS and other core technologies used in the building of most websites. In this unit, you will learn how to use these technologies to develop responsive business and e-commerce websites. Designed with employability in mind, this module will align its learning objectives with skills required within digital agencies and for freelance development work.
